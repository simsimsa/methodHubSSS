# MethodHub

MethodHub — современная цифровая библиотека методических материалов для преподавателей. На сайте собраны пособия, конспекты, планы занятий и другие разработки, аккуратно структурированные по предметам и уровням обучения. Материалы сопровождаются краткими описаниями и удобными тегами, что помогает быстро находить нужную информацию.

## Технологический стек

- **Backend**: NestJS, TypeScript, PostgreSQL
- **Frontend**: React, TypeScript, Vite, Tailwind CSS, Redux Toolkit
- **База данных**: PostgreSQL 15

## Установка и запуск

1. Установите зависимости:
```bash
pnpm install:all
```

2. Запустите базу данных:
```bash
docker-compose up -d
```

3. Запустите проект в режиме разработки:
```bash
pnpm dev
```

Backend будет доступен на http://localhost:3000
Frontend будет доступен на http://localhost:5173

## Структура проекта

- `backend/` - NestJS приложение
- `frontend/` - React приложение
- `database/` - SQL схемы и миграции
